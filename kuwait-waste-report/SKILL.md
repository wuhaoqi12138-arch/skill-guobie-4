# 垃圾量调研报告生成技能

## 概述

本技能用于生成康恒环境海外项目垃圾量调研报告。报告分为**立项版**和**投决版**两种深度，严格遵循康恒环境固定模板结构。立项版用于项目立项阶段快速评估，投决版用于投资决策阶段的深度调研论证。

## 报告模板结构（固定，不可变更）

### 一、立项版模板

```

1.1 调研区域整体概述

    - 气候特征：气候类型、年平均气温、降水分布、年平均降雨量
    - 城市概况：政治经济文化定位、核心产业结构

1.2 人口及财政概况
    - 人口数据：辖区面积、下辖行政单元、近10年户籍/常住人口数量
    - 财政情况：最近年度公共财政收入/支出、人均GDP

    - 运营现状：饱和状态、剩余使用年限
```

## 执行流程

### Step 1: 确认报告版本与参数
- 确认版本：立项版（快速评估）或投决版（深度论证）
- 确认目标国别/城市/服务范围
- 确认项目拟建规模（吨/日）
- 确认用户是否提供实地调研数据

### Step 2: 数据采集
优先级：
1. **用户提供的实地调研数据**（蹲点数据、过磅记录、访谈纪要等）
2. **权威信息源搜索**：
   - 目标国官方统计局 — 人口、经济数据
   - 市政环卫部门 — 垃圾量统计数据、收运体系信息
   - World Bank What a Waste 2.0 — 人均产废量国际对标
   - UNEP固废管理报告 — 区域固废数据
   - 目标国环卫规划文件
3. **科威特知识库参考数据**

### Step 3: 逐章生成（按版本对应模板）
- 立项版：按一至三章结构生成
- 投决版：按一至十二章结构生成，包含附件和承诺书

### Step 4: 垃圾量预测模型（投决版核心）
执行多情景测算：

```
预测公式：年日均垃圾清运量 = 服务人口 × 人均日产生量 × 收运覆盖率 × 修正系数

参数获取：
- 服务人口：基准年人口 + 历史CAGR外推
- 人均日产生量：科威特参考值1.4-1.7 kg/人/日
- 收运覆盖率：根据环卫规划目标设定
- 修正系数：考虑旅游人口季节性、垃圾分类减量等

三种情景输出：
| 情景 | 人口CAGR | 收运覆盖率 | 竞争分流 | 预测结果 |
|------|----------|-----------|---------|---------|
| 保守 | 近7年最低 | 最低规划目标 | 最大分流 | XX t/d |
| 基准 | 近10年平均 | 既定规划目标 | 合理分流 | XX t/d |
| 乐观 | 近20年最高 | 100%覆盖 | 无分流 | XX t/d |
```

### Step 5: 质量校验
- [ ] 模板章节齐全，编号正确
- [ ] 所有表格按模板格式生成
- [ ] 垃圾量数据标注来源和年份
- [ ] 投决版包含多情景预测模型
- [ ] 投决版包含风险识别表
- [ ] 附件清单完整
- [ ] 签字栏保留

### Step 6: 输出报告
- 生成Markdown格式报告
- 立项版保留投资经理/国别代表签字栏
- 投决版保留调研数据真实性承诺

## 注意事项

1. **模板不可变**：章节结构、表格列名必须与 `references/垃圾量调研报告模板.md` 完全一致
2. **数据分离**：官方数据与实测数据需分别记录，偏差需分析说明
3. **预测模型**：投决版必须包含保守/基准/乐观三情景
4. **来源标注**：所有数据括注来源和年份
5. **缺口分析**：基准情景下如存在垃圾量缺口，必须给出应对方案
6. **风险量化**：风险表需量化影响金额
